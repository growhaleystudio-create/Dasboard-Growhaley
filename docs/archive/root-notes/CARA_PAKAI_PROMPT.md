## Struktur Prompt yang Benar

AI carousel generator sekarang sudah diperbaiki untuk memahami struktur prompt dengan benar. Berikut cara penulisan yang tepat:

### ✅ Format yang Disarankan

```
[TOPIK UTAMA]
Deskripsi detail tentang apa yang mau dibahas di carousel

Target Audience: [siapa yang mau kamu target]
Tone: [casual/profesional/edukasi/sales]
Goal: [apa tujuan konten ini]
```

### Contoh 1: Konten Edukasi
```
5 Manfaat Landing Page yang Wajib Kamu Tahu

Bahas kenapa landing page itu penting untuk bisnis online, 
mulai dari konversi lebih tinggi, tracking yang lebih akurat, 
sampai kemudahan A/B testing.

Target Audience: Pemilik bisnis online pemula
Tone: Edukasi, santai tapi informatif
Goal: Edukasi tentang pentingnya landing page
```

**Hasil yang diharapkan:**
- Slide 1: Cover - 5 Manfaat Landing Page
- Slide 2: Manfaat #1 - Konversi Lebih Tinggi
- Slide 3: Manfaat #2 - Tracking Akurat
- Slide 4: Manfaat #3 - A/B Testing Mudah
- Slide 5: Manfaat #4 - [dan seterusnya]
- Slide 10: CTA

**Semua slide membahas TOPIK yang sama**: Manfaat Landing Page

---

### Contoh 2: Product Showcase
```
Kenalan dengan Fitur Baru: AI Content Generator

Showcase fitur terbaru kita yang bisa auto-generate konten 
Instagram carousel dalam hitungan detik. Highlight: 
- Brand kit customization
- Multi-layout options
- One-click export

Target Audience: Digital marketer & content creator
Tone: Excited, modern, tech-savvy
Goal: Product awareness & drive sign-ups
```

---

### Contoh 3: Storytelling
```
Perjalanan UMKM Naik Kelas dengan Digital Marketing

Cerita sukses client kita yang tadinya jualan offline aja, 
sekarang bisa tembus 100 juta/bulan dari online. 
Fokus ke strategi digital marketing yang mereka pakai.

Target Audience: UMKM yang mau scale up
Tone: Inspiratif, relatable, actionable
Goal: Social proof & inspire action
```

## ❌ Kesalahan Umum yang Harus Dihindari

### 1. Menulis Target Audience Sebagai Topik Terpisah
**SALAH:**
```
Manfaat Landing Page
Target Audience
Pemilik Bisnis Online
Call to Action
```
AI akan pikir ini 4 topik berbeda!

**BENAR:**
```
Manfaat Landing Page untuk Pemilik Bisnis Online

Bahas berbagai manfaat punya landing page untuk bisnis...
Target Audience: Pemilik bisnis online pemula
```

### 2. Terlalu Singkat, Kurang Konteks

```
Landing page
```

**LEBIH BAIK:**

```
5 Alasan Kenapa Bisnis Kamu Butuh Landing Page

Target Audience: Pemilik toko online & UMKM
```

**SALAH:**

Biarkan AI yang struktur slidenya!

**BENAR:**
```
Manfaat Landing Page

Bahas 5-7 manfaat utama landing page untuk bisnis online...
```

## 📝 Tips Menulis Prompt yang Efektif

### 1. Mulai dengan Topik Utama yang Jelas
**Satu topik** yang akan dibahas dari berbagai angle di seluruh carousel.

### 2. Berikan Context yang Cukup
- **Target Audience**: Untuk siapa konten ini? (usia, profesi, pain point)
- **Tone**: Bagaimana cara penyampaiannya? (casual, formal, inspiratif, educational)
- **Goal**: Apa tujuan konten? (edukasi, awareness, sales, engagement)

### 3. Berikan Detail yang Relevan
Kalau ada poin-poin spesifik yang mau dibahas, tulis aja. AI akan struktur jadi slide yang koheren.

### 4. Gunakan Bahasa Natural
Tulis seperti kamu brief ke copywriter. Nggak perlu formal banget atau teknis.

## 🎨 Kontrol Layout & Image Style

### Layout: Full AI atau Manual?

AI sudah dilatih untuk **auto-select layout yang tepat** berdasarkan konten. Tapi kamu bisa influence dengan cara:

#### 1. Lewat Kata Kunci di Prompt
AI akan detect keyword tertentu dan adjust layout style:

**Editorial/Magazine Style:**
```
Buat carousel bergaya editorial majalah tentang [topik]

Keywords: editorial, majalah, reportage, profile, opini
Hasil: AI akan prioritaskan layout editorial (pullquote, column, feature spread)
```

**Information Dense:**
```
Panduan lengkap 10 langkah [topik]

Keywords: langkah, checklist, panduan, step-by-step
Hasil: Layout dengan numbered steps, checklist, comparison table
```

**Visual Heavy:**
```
Showcase produk dengan fokus visual

Keywords: showcase, portfolio, gallery, before-after
Hasil: Layout image-focused dengan dual image, gallery, comparison
```

#### 2. Reference Mode (di UI Dashboard)

Ada 3 mode yang bisa kamu pilih:

**Mode: Otonom** (Default)
- AI full control pilih layout
- Cocok untuk: Konten rutin, trust AI sepenuhnya

**Mode: Auto Match**
- AI pilih dari katalog reference yang udah kamu approve
- System akan match prompt dengan reference paling relevan
- Cocok untuk: Konsistensi brand, punya style guide ketat

**Mode: Manual**
- Kamu pilih exact reference yang mau diikuti
- AI akan copy layout structure dari reference itu
- Cocok untuk: Recreate style tertentu, konsistensi campaign

### 💡 Tips Menggunakan Image Style

1. **Satu Style Per Carousel**
   - Konsisten gunakan satu style untuk semua slide
   - Jangan mix "doodle" slide 1 dengan "photo" slide 2

2. **Sebutkan Style di Awal Prompt**
   ```
   [Topik] dengan visual style [nama style]
   
   Gambar: [deskripsi spesifik style]
   ```

3. **Kalau Tidak Specify, AI Pakai Default**
   - Default: Clean professional editorial illustration
   - Safe choice untuk most cases

4. **Combine dengan Brand Colors**
   - Image style akan adapt dengan Brand Kit colors kamu
   - Flat vector & doodle paling flexible dengan brand colors

## 💡 Yang Perlu Diingat

1. **Satu Topik = Satu Carousel**
   - Semua slide bahas topik yang sama dari angle berbeda
   - Jangan campur beberapa topik tidak related

2. **Target Audience = Konteks, Bukan Topik**
   - Target audience menentukan **cara penyampaian** dan **bahasa**
   - Bukan jadi slide terpisah

3. **AI Butuh Context, Bukan Script**
   - Tulis ide besar dan poin-poin penting
   - Biarkan AI yang craft copywriting & struktur slide

4. **Conversation Context Matters**
   - Chat sebelumnya jadi context untuk request berikutnya
   - Kalau mau topik baru yang beda total, lebih baik start fresh

## 🔧 Fitur Planning Mode

Kalau kamu aktifkan **Planning Mode** (toggle di bawah textarea):
- AI akan bikin draft dulu sebelum generate visual
- Kamu bisa review & revisi struktur slide
- Lebih cocok untuk konten kompleks atau campaign penting

Kalau **Planning Mode OFF**:
- AI langsung generate full carousel
- Lebih cepat, cocok untuk konten rutin atau iterasi cepat

---

## ❓ Pertanyaan?

Kalau hasil generate belum sesuai:
1. Cek apakah prompt structure sudah jelas (topik utama, audience, goal)
2. Coba tambahkan lebih banyak context/detail
3. Gunakan Planning Mode untuk review struktur dulu
4. Manfaatkan fitur "Revise" untuk perbaiki tanpa generate ulang dari awal

## 🎛️ Quick Reference: Kontrol di Dashboard

### Config yang Tersedia:

1. **Aspect Ratio**: 1:1 (Instagram) atau 9:16 (Story)
2. **Slide Count**: Jumlah slide (default: AI decide optimal)
3. **Header/Body Size**: Typography scale (editorial_bold / balanced / dense)
4. **Tags**: Custom tags untuk content organization
5. **Reference Mode**: Otonom / Auto Match / Manual
6. **Planning Mode**: Draft dulu atau langsung generate

### Prioritas Kontrol:

**Prompt > Config > AI Decision**

- Kalau kamu specify di prompt (misal "10 slide"), prompt wins
- Kalau config set tapi prompt tidak mention, config wins  
- Kalau keduanya silent, AI decide based on content complexity
